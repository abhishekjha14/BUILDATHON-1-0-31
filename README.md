# PhishNet Sentinel
PhishNet Sentinel – AI-Driven Multi-Channel Phishing Detection & Prevention System
An advanced, real-time phishing defense platform that protects users across emails, SMS, voice calls, QR codes, and malicious websites using cutting-edge AI/ML and blockchain-powered immutable logging.
Key Features

Multi-channel threat detection (Email, SMS, Voice Phishing/Vishing, QR codes, URLs)
Real-time analysis powered by NLP, computer vision, and behavioral modeling
Centralized AI Core that fuses intelligence from specialized analysis modules
Blockchain Ledger for tamper-proof logging and provenance of detected threats
Adaptive learning to counter evolving attack techniques
User education & awareness integration

Tech Stack

Python | TensorFlow / PyTorch | OpenCV
AWS Cloud Infrastructure
Blockchain (immutable threat logging)
Speech-to-text & voice analysis models
QR code & image forensic analysis

Why PhishNet Sentinel?
In a world where phishing causes billions in annual losses and attacks are getting increasingly sophisticated, traditional filters are no longer enough. PhishNet Sentinel acts as an intelligent, proactive shield that detects and blocks threats before they reach the user — across all digital touchpoints.
